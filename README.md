# Queue Waiting Line System (C++)

A C++ implementation of a queue-based waiting line system that simulates ticket issuance and client servicing, demonstrating the Queue data structure through a real-world waiting line scenario.

### Features

- Ticket generation system
- Client queue management
- Waiting time estimation
- Queue visualization
- Client servicing simulation
- First-In First-Out (FIFO) processing

### Concepts Practiced

- Queue Data Structure
- FIFO (First In, First Out)
- Queue Operations
- Data Structure Applications
- System Simulation
- Object-Oriented Programming (OOP)

### Technologies

- C++
- Standard Template Library (STL)
- Console Application

## Purpose

This project was developed to demonstrate a practical real-world application of the Queue data structure by simulating a customer waiting line system similar to those used in banks, hospitals, and service centers.
